## Software

For this workshop, we will use the following tools:

* Quality control: `fastqc`
* Trimming: `fastp`
* Mapping: `STAR` (downloaded [here](https://github.com/maschon0/scPlants_practical0/tree/main/software))
* Counting: `featureCounts` 
* SAM file handling: `samtools` (downloaded [here](https://github.com/maschon0/scPlants_practical0/tree/main/software))
