# RNA-seq-workshop-materials

This repository contians the materials needed for the bioDSC workshop "RNA-seq analysis". The materials are used in a Google Colab notebook, where we use a Virtual Machine to run bioinformatics commands on the data. Find the Google Colab [here](https://colab.research.google.com/drive/1jR2Og_qrUVg-VXNsfqJc5eEILcqHJYAC)
